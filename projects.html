PickMeUp

This is a phone app designed to aid visually impaired people locate an object in a room, walk towards it, and pick it up. The phone is placed onto a bracket on a glove with vibration motors. With the use of object detection through a neural network and augmented reality, the user scans the room, which locates all compatible objects. The user then tells the app what he or she wants to pick up, and the app guides the user towards said object with the vibration motors, avoiding any obstacles.

SecureSafe

A password organizer app designed to securely save passwords for multiple user accounts and occasions. Uses SHA-1 cryptographic hashing to ensure all passwords are secure and unable to be stolen from malware.
